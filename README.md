# Online Learning Management System

A comprehensive **Learning Management System (LMS)** designed to facilitate online education. Built with **PHP** and **MySQL**, this platform allows instructors to create and manage courses while providing students with a seamless learning experience through video lessons and interactive features.

## 🚀 Features

### **Admin Panel**
- **Dashboard:** At-a-glance view of platform metrics (Total Courses, Students, Sold Courses).
- **Course Management:** capabilities to Add, Edit, and Delete courses, including details like price, description, and cover images.
- **Lesson Management:** Organize curriculum by adding video lessons to specific courses.
- **Student Management:** View registered students and manage user accounts.
- **Sales Reports:** Generate and view reports on course sales and revenue.
- **Payment Verification:** Check and verify payment statuses.
- **Feedback System:** Review feedback and testimonials submitted by students.
- **Security:** secure login and password management for administrators.

### **Student Panel**
- **User Profile:** Students can manage their personal information.
- **My Courses:** Access to enrolled courses and video content.
- **Feedback:** Option to provide reviews and feedback on courses.
- **Security:** Secure login and password change functionality.

### **Public Interface**
- **Responsive Landing Page:** Engaging homepage with video background, popular courses, and student testimonials.
- **Course Catalog:** Browse and search through available courses.
- **Registration & Login:** Secure authentication system for students.
- **Contact:** Integrated contact form for inquiries.

## 🛠 Technologies Used

- **Frontend:**

  - JavaScript & jQuery

- **Backend:**
